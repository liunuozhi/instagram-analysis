# instagram-analysis

Try to analyze instagram in urban context.

## Getting Started

Custom stop word into ```./notebook/add_stopword.txt```. See more info in [jieba](https://github.com/fxsjy/jieba).

Also, I am working on Windows 10 environment.

### Prerequisites

```
anaconda python=3.7
```

### Installing

```
pip install emoji
pip install jieba
pip install pyproj
```
